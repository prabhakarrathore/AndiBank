End-User Module Features

The ANDi Banking platform provides end-users with a secure and convenient online banking experience. Key features available to customers include:

Account Access & Management

  • Secure Login: User authentication via username and password (Optional: Social Login via Facebook if configured).
  • User Dashboard: A personalized overview displaying account balances, recent transactions, and quick links.
  • Profile Management: Ability for users to view and update their contact information and security settings (e.g., password change).
  • Account Statements: View and download transaction history for selected periods.

Financial Transactions

  • Fund Deposits: Users can add funds to their accounts using integrated payment methods (configured by admin, e.g., PayPal, Card Processors, Bank Transfer details).
  • Fund Withdrawals: Request withdrawal of funds to linked external accounts or via other methods enabled by the administrator.
  • Internal Transfers: Transfer funds instantly between user accounts within the ANDi Banking system.
  • External Transfers: Initiate transfers to accounts at other banks (if module is configured).
  • Transaction History: Detailed log of all deposits, withdrawals, and transfers made by the user.

Information & Communication

  • News & Announcements: View latest updates published by the bank administrators.
  • FAQ Section: Access frequently asked questions for self-service support.
  • Notifications: Receive system alerts related to account activity or security (via platform, potentially email/SMS if configured).
  • Contact Information Display: View bank's contact details (phone, email, address) managed by the admin.

Note: The exact availability and appearance of features depend on the specific configuration set by the bank administrator.
